# Todo Application

## Project Description
This is a simple Todo application that allows users to add, manage, and delete tasks. 
The application helps users organize daily activities efficiently through a clean and interactive interface.

## How to Run Locally
Follow these steps to run the project on your local machine:

1. Clone the Repository
git clone https://github.com/latifurrafi/Ostad_batch-09.git

2. Navigate into the Project Folder
cd Ostad_batch-09

3. Install dependencies:
pip install -r requirements.txt

4. Apply migrations:
python manage.py migrate

5.Run the development server:
python manage.py runserver

6.Open in browser:
http://127.0.0.1:8000/
